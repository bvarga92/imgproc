# Programok az OpenCV könyvtárral

A kódok az OpenCV 3.1.0 verziójával készültek.

- **corners_lines**: sarkok keresése és egyenesek illesztése
- **motion.cpp**: egyszerû valós idejû mozgásdetektálás
- **object_tracking.cpp**: valós idejû tárgykövetés színfelismerés alapján ([videó](https://www.youtube.com/watch?v=49DFy6hx-Uk&list=PL9_VlVdB8s882QMHiqJlDpJeKWxwP5CIG))
- **webcam_filter.cpp**: lineáris szûrõk futtatása valós idõben, webkamera képén
